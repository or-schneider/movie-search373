# Movie search
React implementation of a movie search app

- Saves queries in html path
- Pagination between search results
- In mem cache at server side
- Responsive

Deployed at
```  
https://movies-search373.herokuapp.com/
```
## Installation
### Server
```
cd server
npm install
```
### Client
```
cd client
npm install
```
## Usage
### Server
```
cd server
npm run dev
```
### Client
```
cd client
npm run dev
```
